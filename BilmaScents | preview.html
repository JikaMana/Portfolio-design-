Project Overview

Bilma Scents is an ecommerce web application I developed to provide a stylish and user-friendly platform for showcasing and selling luxury perfumes and fragrance products. The platform is designed to support a smooth shopping experience, allowing customers to browse new arrivals, explore product collections, and join waitlists for upcoming stock. I focused on creating a modern interface with clean design elements, interactive product displays, and a responsive layout that adapts well to mobile and desktop screens. By using React.js, Tailwind CSS, and Embla Carousel, I was able to build a fast, visually engaging frontend with custom components that emphasize brand identity and usability. The project includes features like a dynamic product grid, animated transitions, and a custom-built "New Arrivals" carousel to highlight the latest offerings. This project allowed me to deepen my skills in building ecommerce solutions, optimizing user interfaces for real-world businesses, and delivering a production-ready digital storefront for an emerging perfume brand.
